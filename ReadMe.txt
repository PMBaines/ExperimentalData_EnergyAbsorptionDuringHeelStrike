Experimental data and processing scripts (Matlab) corresponding to manuscript 'Experimental estimation of energy absorption during heel strike in human barefoot walking' by Baines, Schwab and van Soest.

If you do not have a Matlab license you can find information on the raw data by opening getRawDataStructure.m as a text file.

If you do have a Matlab lisence you can run plotEnergyLossEstimates and all the files will be loaded, the data will be processed and the figures will be created as shown in the manuscript.
